### 문제 설명

> 1부터 입력받은 숫자 n 사이에 있는 소수의 개수를 반환하는 함수, solution을 만들어 보세요.
> <br>
> 소수는 1과 자기 자신으로만 나누어지는 수를 의미합니다.
> (1은 소수가 아닙니다.)

### 제한 조건

> n은 2이상 1000000이하의 자연수입니다.

### 입출력 예

| n   | result |
| --- | ------ |
| 10  | 4      |
| 5   | 3      |

### 나의 풀이

```js
function solution(n) {
  const arr = [];
  for (let i = 2; i <= n; i++) {
    arr[i] = i;
  }
  //배열에 반복문 적용
  //[2,3,4,5,6,7,8,9,10]
  for (let i = 0; i <= arr.length; i++) {
    if (arr[i] !== 0) {
      for (let j = arr[i] * 2; j <= arr.length; j += arr[i]) {
        arr[j] = 0;
      }
    }
  }
  return arr.filter(el => el !== 0).length;
}

//소수는 1과 자기 자신으로만 나누어지는 수.
//아리스토텔레스의 체
//2의 배수, 3의 배수, 5의 배수 순서대로 삭제.
```

- 어제부터 풀려고 고민하다가 해결책을 못찾아 검색해보니 소수를 구하는 방식 중 **"[아리스토텔레의 체](https://ko.wikipedia.org/wiki/%ec%97%90%eb%9d%bc%ed%86%a0%ec%8a%a4%ed%85%8c%eb%84%a4%ec%8a%a4%ec%9d%98_%ec%b2%b4)"**라는 방식이 있었다.
  처음에는 모든 숫자를 나눠주는 방식으로 하려고 했지만, 범위가 늘어날수록 효율성이 떨어지는것을 발견했고 이 공식을 사용하게 되었다.
- 방식은 2부터 n까지의 숫자를 배열에 나열하고 2부터 자신의 배수를 삭제해가는 방식으로 결국 남는건 소수들만 남게 된다.

1. 먼저 배열에 2부터 n까지의 숫자를 해당하는 인덱스에 넣는 작업을 해줬다.
2. 2중 반복문을 사용해서 각 숫자의 배수를 0으로 바꿔주었다.
3. filter 메서드를 사용해 배열 중 값이 0이 아닌 숫자들만 새로운 배열로 만들어 길이를 반환했다.
