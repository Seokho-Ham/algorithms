### 문제 설명

> 수많은 마라톤 선수들이 마라톤에 참여하였습니다.
> <br>
> 단 한 명의 선수를 제외하고는 모든 선수가 마라톤을 완주하였습니다.
> <br>
> 마라톤에 참여한 선수들의 이름이 담긴 배열 participant와 완주한 선수들의 이름이 담긴 배열 completion이 주어질 때, 완주하지 못한 선수의 이름을 return 하도록 solution 함수를 작성해주세요.

### 제한사항

> 1. 마라톤 경기에 참여한 선수의 수는 1명 이상 100,000명 이하입니다.
>    <br>
> 2. completion의 길이는 participant의 길이보다 1 작습니다.
>    <br>
> 3. 참가자의 이름은 1개 이상 20개 이하의 알파벳 소문자로 이루어져 있습니다.
>    <br>
> 4. 참가자 중에는 동명이인이 있을 수 있습니다.

### 나의 풀이

#### Javascript를 이용한 풀이

```js
function solution(participant, completion) {
  let p = participant.sort();
  let c = completion.sort();

  for (let i = 0; i < p.length; i++) {
    if (p[i] !== c[i]) {
      return p[i];
    }
  }
}
```

- 각 목록을 정렬하여 순서대로 비교했다. 동일한 값이 나오지 않으면 완주하지 못한 사람이기 때문에 그 값을 리턴했다.

#### Java를 이용한 풀이

```java
import java.util.*;

//1번 - sort를 사용해서 풀이

class Solution{
    public String solution(String[] participant, String[] completion) {

        Arrays.sort(participant);
        Arrays.sort(completion);

        int i;
        for(i=0; i<completion.length; i++){
            if(!participant[i].equals(completion[i])){
                return participant[i];
            }
        }
        return participant[i];
    }
}

//2 - 다른 분의 풀이를 보고 이해한 코드
class Solution {
    public String solution(String[] participant, String[] completion) {
        String answer = "";
        HashMap<String, Integer> hashmap = new HashMap<>();

        for(String name : participant){
            hashmap.put(name, hashmap.getOrDefault(name,0)+1);
        }
        for(String name : completion){
            hashmap.put(name, hashmap.get(name)-1);
        }

        for(String key : hashmap.keySet()){
            if(hashmap.get(key)!=0){
                answer = key;
                break;
            }
        }
        return answer;
    }
}
```

- java를 학습한 이후 java문법을 이용해 프로그래머스의 문제들을 다시 풀어보고 있다.
- Hashmap을 사용한 풀이는 기존에 작성한 코드의 효율성이 좋지 않은것같아서 다른분의 풀이를 보다가 발견했다.
