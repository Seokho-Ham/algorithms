### 문제 설명

> 단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요.
> <br>
> 단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.

### 제한사항

> s는 길이가 1 이상, 100이하인 스트링입니다.

### 나의 풀이

```js
function solution(s) {
  let mid = Math.floor(s.length / 2);

  if (s.length % 2 === 0) {
    return s[mid - 1] + s[mid];
  }

  return s[mid];
}
```

1. 문자열의 중간값을 구하고 짝수일경우 그 앞의 단어와 함께 반환하도록 작성했다.
   Math.floor 메서드는 파라미터로 들어온 수보다 작은 수 중 가장 큰 정수를 반환한다.
